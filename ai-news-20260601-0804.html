Anthropic 宣布 6 月 15 日重大计费变革：Agent SDK、Claude Code GitHub Actions 等程序化用量从订阅额度剥离，移入独立信用池，按 API 标准定价计费

6 月 15 日起，Anthropic 将对 Claude 订阅计费结构进行有史以来最大规模的重构——将"程序化/代理式用量"从用户订阅配额中剥离，引入独立的"Agent SDK 信用池"。受影响范围：Claude Agent SDK（包括 -p 模式）、Claude Code GitHub Actions 自动化工作流、通过 Anthropic API 驱动的第三方代理，以及所有非交互式批量处理场景。不受影响范围：claude.ai 网页/桌面/移动端交互聊天、Claude Code 在终端内的交互式使用、Claude Cowork 协作模式——上述场景继续沿用原有订阅配额，无需任何调整。信用池金额：Pro 订阅 $20/月；Max 5x 订阅 $100/月；Max 20x 订阅 $200/月，全部按 API 标准列表定价计量，无余额滚存。成本影响：过去 Claude 订阅将代理用量的实际成本补贴约 15–30 倍；新方案移除补贴后，重度程序化用户的月度成本将显著上升，需在 6 月 15 日前完成"领取信用"动作并决定是否开启"用量溢出"选项（溢出按 API 定价继续扣费，否则请求被拒绝）。开发者社区反应：大量 indie hacker 和企业开发者表示需要重新评估代理工作流的运营成本模型，部分用户已开始探索自托管方案或迁移至直接 API 调用以优化成本；Zed、Cursor 等 IDE 集成商已发布专项公告说明影响范围。分析师指出，此举是 Anthropic 在冲击 IPO 前提升商业可预测性的重要一步，也标志着"AI 代理基础设施"的定价模式正式告别消费者订阅的交叉补贴时代。

5 月下旬，OpenAI 正式发布 Rosalind Biodefense 项目——面向经过审查的开发者、美国政府合作机构和公共卫生研究机构，开放专为生物安全应用场景定制的 GPT-Rosalind 模型受信访问（Trusted Access）。命名背景：Rosalind 得名于 DNA 双螺旋结构发现中至关重要的科学家 Rosalind Franklin，延续 OpenAI"以女性科学先驱命名专业垂直模型"的命名传统（此前包括 GPT-4.5 生物医学版本）。主要能力：① 基因组序列分析——快速识别已知和新型病原体基因特征，辅助早期威胁评估；② 大流行病传播建模——基于流行病学数据实时构建传播预测模型，辅助政策制定；③ 医学对策研发支持——快速综述相关文献，加速疫苗候选靶点筛选和药物再利用研究；④ 生物安全文件分析——自动解析实验室安全规程、合规文件和国际生物安全条约文本。访问机制：申请机构须通过 OpenAI 受信合作伙伴审查流程，包括机构背景核实、使用目的声明和数据安全协议签署；非政府研究机构须提供学术隶属证明和 IRB/伦理审查材料。政策背景：这是 OpenAI "扩展受信访问"战略的最新一步，此前已向网络安全领域（GPT-5.5-Cyber）开放类似机制；两者共同体现了 OpenAI 将前沿 AI 能力有条件地引入高风险但高社会价值领域的审慎商业化路径。批评者指出，此类"受信访问"机制的边界界定和监督机制仍需更大透明度，以防止能力被滥用于双用途（Dual-Use）场景。
