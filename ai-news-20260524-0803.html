Anthropic · SDK 战略 收购 Stainless · $3 亿+
Anthropic 以逾 $3 亿收购 Stainless：截断 OpenAI 与 Google 共用的 SDK 关键基础设施

Anthropic 宣布完成对 SDK 自动生成初创公司 Stainless 的收购，The Information 报道交易金额超过 $3 亿（约合 22 亿人民币）。Stainless 由前 Stripe 工程师 Alex Rattray 于 2022 年创立，其核心产品是将 API 规范自动转化为多语言生产级 SDK——支持 Python、TypeScript、Kotlin、Go、Java 等主流语言——被 OpenAI、Google、Cloudflare、Merge 等顶级 AI 公司广泛依赖，用以生成和维护各自的官方 SDK。Anthropic 表示，Stainless 从创立之初就为 Anthropic 的所有官方 SDK 提供支持。收购完成后，Anthropic 宣布将关闭所有 Stainless 托管产品，包括其公开的 SDK 生成器服务，现有客户保留已生成 SDK 的完整所有权和修改权。这一动作的战略意图十分清晰：将一个竞争对手高度依赖的关键开发者基础设施纳入自身生态，使其他 AI 公司必须寻找替代方案或自行维护 SDK 生成管线，同时也让 Anthropic 在开发者工具体验和 SDK 质量上获得独特的内部优势。开发者社区对此普遍认为，这是 Anthropic 从"模型提供商"向"全栈开发者平台"深度转型的重要信号，与 Claude Code 崛起、MCP 标准化等举措共同构成一条完整的开发者生态护城河战略。
